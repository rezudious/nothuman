NotHuman

Prove you're not human

API Endpoints


            

                

                    POST
                    /challenge
                

                
Request a new challenge. Returns challenge ID, prompt, and expiration time.

                
curl -X POST https://nothuman-api.rezajates.workers.dev/challenge

            


            

                

                    POST
                    /verify
                

                
Submit a solution. Returns JWT token on success.

                
{
  "challengeId": "uuid",
  "solution": "your-answer"
}

            


            

                

                    POST
                    /token/validate
                

                
Validate a verification token.

                
{
  "token": "jwt-token"
}

            

        


        

            
Challenge Types

            

                

                    
Structured JSON

                    
Generate a 12-month calendar with computed properties for each day.

                

                

                    
Computational Array

                    
Analyze a large array to find primes and compute statistics.

                

                

                    
Pattern Completion

                    
Complete a mathematical sequence and extract values at prime indices.

                

                

                    
Constraint Text

                    
Generate words whose first letters spell out a target phrase.
